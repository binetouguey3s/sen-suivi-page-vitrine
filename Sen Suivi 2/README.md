# 🧠 Sen Suivi – Bien-être Étudiant

## 📋 Description du Projet

Plateforme de sensibilisation à la santé mentale des étudiants, développée dans le cadre de l'Atelier 4 : HTML – CSS (BTS Informatique Industrielle et Réseaux).

## 🎯 Objectifs de la Plateforme

- Sensibiliser les étudiants sur la santé mentale
- Prévenir le stress, l'anxiété et l'isolement
- Encourager le bien-être émotionnel et social

## 📁 Structure du Projet

```
/Sen Suivi 2
├── index.html       # Page principale (HTML5 sémantique)
├── style.css        # Feuille de styles (CSS3)
├── images/          # Dossier pour les images
└── README.md        # Ce fichier
```

## 🚀 Comment utiliser le projet

1. **Ouvrir le projet** : Double-cliquez sur `index.html` dans votre navigateur
2. **Ajouter des images** : Placez vos images dans le dossier `images/`
3. **Modifier le contenu** : Éditez `index.html` pour personnaliser le texte
4. **Modifier le design** : Éditez `style.css` pour changer les couleurs et styles

## ✨ Fonctionnalités Implémentées

### HTML5 Sémantique
- ✅ Utilisation de balises sémantiques (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`)
- ✅ Structure claire et logique
- ✅ Attributs `alt` pour l'accessibilité
- ✅ Liens de navigation internes (ancres)
- ✅ Formulaire de contact fonctionnel

### CSS3 Moderne
- ✅ **Flexbox** pour la mise en page responsive
- ✅ **Gradient** pour un design moderne
- ✅ **Transitions et animations** pour l'interactivité
- ✅ **Media queries** pour le responsive design
- ✅ Palette de couleurs apaisante (violet, bleu)

### Sections Obligatoires
1. ✅ **En-tête** : Logo et menu de navigation
2. ✅ **Section À propos** : Mission, objectifs, importance
3. ✅ **Section Services** : 4 cartes de services
4. ✅ **Section Contact** : Formulaire (nom, email, message)
5. ✅ **Pied de page** : Copyright et réseaux sociaux

## 🎓 Points Pédagogiques à Présenter

### 1. HTML5 Sémantique
Expliquez pourquoi vous avez utilisé :
- `<header>` pour l'en-tête plutôt que `<div>`
- `<nav>` pour la navigation
- `<section>` pour grouper le contenu thématique
- `<article>` pour les cartes de services

### 2. CSS3 Moderne
Expliquez comment vous avez utilisé :
- **Flexbox** pour aligner les éléments (navigation, grille de services)
- **Gradient linéaire** pour le header et le hero
- **Box-shadow** pour les effets de profondeur
- **Transitions** pour les animations au survol

### 3. Bonnes Pratiques
Mentionnez :
- Aucun style inline (tout dans `style.css`)
- Code commenté pour faciliter la maintenance
- Respect de l'accessibilité (labels, alt, aria-label)
- Design responsive (mobile-first)
