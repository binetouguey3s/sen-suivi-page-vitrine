# 🎓 CyberFormation Pro - Page Vitrine

## 📋 Description du projet

Page vitrine professionnelle créée dans le cadre de l'Atelier 4 : HTML - CSS.
Ce projet présente une plateforme de formation en cybersécurité et réseaux informatiques.

## 🎯 Objectifs pédagogiques atteints

✅ HTML5 sémantique (header, nav, section, article, footer)
✅ Aucun style inline
✅ Aucune balise obsolète
✅ CSS externe dans un fichier séparé
✅ Navigation avec ancrages internes
✅ Formulaire de contact fonctionnel
✅ Design responsive (mobile-first)
✅ Utilisation de Flexbox et Grid
✅ Code commenté et lisible

## 📁 Structure du projet

```
page-vitrine/
├── index.html          # Page principale
├── style.css           # Feuille de styles
├── images/             # Dossier pour les images
│   └── cybersecurity.jpg (à ajouter)
```

## 🖼️ Images à ajouter

**Nom du fichier :** `cybersecurity.jpg`

Vous pouvez télécharger une image libre de droits sur :
- Unsplash.com

- Pixabay.com

1. Assurez-vous que tous les fichiers sont dans le bon dossier
2. Ajoutez votre image dans le dossier `images/`
3. Ouvrez `index.html` dans votre navigateur

## 📱 Fonctionnalités

### Navigation
- Menu fixe en haut de page
- Liens d'ancrage vers chaque section
- Effet hover sur les liens

### Sections
1. **Présentation** : Description de la plateforme avec image
2. **Formations** : 4 cartes présentant les différentes formations
3. **Contact** : Formulaire avec nom, email et message
4. **Footer** : Informations de contact et copyright

### Responsive Design
- Adapté aux mobiles (< 480px)
- Adapté aux tablettes (< 768px)
- Adapté aux ordinateurs (> 768px)

## 🔧 Technologies utilisées

- **HTML5** : Balises sémantiques
- **CSS3** : Flexbox, Grid, Variables CSS, Media Queries
- **Aucun framework** : Code vanilla pour mieux comprendre les bases

## 📝 Points à expliquer lors de la présentation

1. **Pourquoi HTML sémantique ?**
   - Meilleure accessibilité
   - Meilleur référencement SEO
   - Code plus lisible

2. **Avantages du CSS externe**
   - Séparation contenu/présentation
   - Réutilisabilité
   - Maintenance facilitée

3. **Flexbox vs Grid**
   - Flexbox : navigation, formulaire (une dimension)
   - Grid : cartes de formations (deux dimensions)

4. **Variables CSS**
   - Centralisation des valeurs
   - Maintenance simplifiée
   - Cohérence visuelle

5. **Mobile-first**
   - Adaptation progressive
   - Meilleure expérience utilisateur
   - Design moderne
